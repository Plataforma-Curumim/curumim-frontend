Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de facilitar o gerenciamento de livros em bibliotecas utilizando tecnologia RFID. O sistema identifica rapidamente o usuário e o livro, otimizando o processo de empréstimo e devolução.

Equipe

O projeto foi realizado pela equipe da Arandu Technology, uma startup focada em soluções inovadoras para o setor de tecnologia educacional e de gestão.
